browser setup

firefox

pre-requisites

  • download userChrome.css:
    • userChrome.css
  • install sidebery
    • sidebery (firefox addon store)
  • import sidebery config or setup manually
    • sidebery.json

setup

  1. open "about:profiles" and click "Open Folder" next to profiles root directory
  2. create new folder named "chrome", and add "userChrome.css" to the new folder
  3. open "about:config" and enable "toolkit.legacyUserProfileCustomizations.stylesheets"
  4. restart firefox
